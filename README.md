🗂️ Itinerary / Task Manager

A simple and modern task management web application built with Next.js.
It helps users organize their daily activities by creating tasks, setting a date/time, and tracking completion status.

The interface focuses on clarity, productivity, and a clean dark UI design.

✨ Features

➕ Add tasks with a date and time

✅ Mark tasks as completed

📌 Filter tasks by:

All

Pending

Completed

🗑️ Delete individual tasks

🧹 Clear all tasks

📊 Live task statistics (Total / Pending / Completed)

🌙 Clean dark theme interface

# or

# or
pnpm dev
# or

Open the application in your browser:
